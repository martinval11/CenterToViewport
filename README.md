# CenterToViewport

A JavaScript library for centering an element to the viewport.

## How to use it?

First, install it using `npm`:

```sh
npm install ctv
```

```js
import { CenterToViewport } from 'ctv';

function App() {
  return (
    <CenterToViewport>
      <div>
        <h1>This is centered</h1>
      </div>
    </CenterToViewport>
  );
}
```
